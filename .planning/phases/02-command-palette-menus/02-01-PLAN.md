    - handle_command(repl, "/exit") and "/quit" -> CommandResult("exit")
    - handle_command(repl, "/help") -> action "handled", render is the _help_table()
    - handle_command(repl, "/models") -> routes to the /models handler (NOT /model)
    - handle_command(repl, "/model ollama:llama3.1") -> switches model (arg="ollama:llama3.1")

    - handle_command(repl, "/mode auto") -> sets mode; "/mode" bare -> cycles
    - handle_command(repl, "/unknowncmd") -> "handled" with "unknown command ..."
    - handle_command(repl, "hello world") -> CommandResult("turn")
  </behavior>
  <action>
    Add at module top (after CommandResult):

    ```python
    from typing import Callable, Literal, Optional
    ChoiceKind = Literal[None, "model", "mode"]

    @dataclass(frozen=True)
    class CommandEntry:
        name: str                    # e.g. "/model" (no trailing space)
        usage: str                   # e.g. "/model <provider:model>"
        description: str             # shown in palette + /help
        handler: Callable[[Any, str], CommandResult]
        needs_choice: ChoiceKind = None
        terminal: bool = False
    ```

    Define `COMMANDS: list[CommandEntry]` with one entry per existing command, in the SAME order
    as the current `_help_table` rows. Each handler is `lambda repl, arg: CommandResult(...)`
    reproducing the exact current branch body:
    - "/help" -> CommandResult("handled", _help_table())
    - "/config" -> CommandResult("handled", repl.cfg.model_dump_json(indent=2))
    - "/models" -> CommandResult("handled", _models_tables(repl.cfg))  # returns the (providers, roles) tuple
    - "/model" usage "/model <provider:model>" needs_choice="model" -> CommandResult("handled", _set_model(repl, arg))
    - "/tools" -> CommandResult("handled", "tools: " + ", ".join(repl.harness.registry.names()))
    - "/skills" -> CommandResult("handled", <the existing skills-render string>)
    - "/memory" -> CommandResult("handled", _show_memory(repl))
    - "/mode" usage "/mode [ask|auto|deny]" needs_choice="mode" -> CommandResult("handled", _set_mode(repl, arg))
    - "/cost" -> CommandResult("handled", _show_cost(repl))
    - "/clear" -> handler that sets repl.state = repl.harness.new_session() then returns CommandResult("handled", "conversation cleared")
    - "/exit" terminal=True -> CommandResult("exit")

    Add `def find_command(name: str) -> CommandEntry | None: return next((c for c in COMMANDS if c.name == name), None)`.

    Rewrite `handle_command` to (in order): empty -> noop; `line in ("/exit", "/quit")` -> exit
    (KEEP the /quit alias — it is NOT a COMMANDS entry); not startswith("/") -> turn;
    `name, _, arg = line.partition(" ")`; `entry = find_command(name)`; if None ->
    CommandResult("handled", f"unknown command {line}"); else `return entry.handler(repl, arg.strip())`.

    Extend `__all__` to include "CommandEntry", "COMMANDS", "find_command".
